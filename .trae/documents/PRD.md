# 二次元摄影师预约展示网站 - 产品需求文档

## 1. 产品概述

一个面向二次元摄影的摄影师预约展示网站，访客可以浏览摄影师作品例图、查看并提交拍摄预约，摄影师可在后台管理作品和预约。系统基于 Next.js + Supabase 构建，支持深色模式和响应式布局。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览作品、提交预约 |
| 摄影师 | 邮箱注册 | 登录后台、管理作品、管理预约 |

### 2.2 功能模块

1. **作品展示**：作品网格展示、图片轮播、分类筛选、作品详情
2. **预约系统**：预约表单、日期时间选择、备注提交、预约确认
3. **后台管理**：作品上传、作品管理、预约列表、预约状态更新
4. **主题系统**：深色/浅色模式切换

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 作品画廊 | 作品网格展示、分类筛选 |
| 作品详情 | 大图展示 | 单张作品大图、作品信息 |
| 预约页面 | 预约表单 | 姓名、联系方式、日期时间选择、备注 |
| 后台登录 | 登录 | 摄影师登录 |
| 后台首页 | 概览 | 预约和作品统计 |
| 作品管理 | 作品列表 | 上传、编辑、删除作品 |
| 预约管理 | 预约列表 | 查看预约、更新状态、删除预约 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：粉紫色渐变（#a855f7 → #6366f1），营造二次元梦幻感
- **强调色**：紫色/粉色渐变按钮、霓虹光晕特效
- **按钮样式**：圆角柔和，悬停时粉色光晕动效
- **字体**：标题使用 Ysabeau SC（复古衬线，正文使用 Inter 无衬线
- **布局风格**：作品画廊式网格，卡片式展示
- **图标**：lucide-react 图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 作品画廊 | 响应式网格、毛玻璃效果卡片、渐变装饰边框、作品悬停放大动效 |
| 作品详情 | 大图展示 | 居中大图查看、作品信息卡片、返回画廊 |
| 预约页面 | 预约表单 | 渐变边框、日期时间选择器、柔和圆角输入框 |
| 后台管理 | 管理面板 | 侧边栏导航、内容区表格 |

### 4.3 响应式

- 桌面优先，最大 1280px 宽
- 移动端 1024px 以下自适应
- 640px 以下堆叠布局

### 4.4 深色模式

- 使用 Tailwind `dark:` 类
- 深色背景粉紫渐变底色
